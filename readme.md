# Урок по созданию приложения списка покупок #
#### Корзина для продуктов с использованием React JS ####
Сайт проекта: [React Shopping Cart](http://adel-ismagilov.ru/lessons/basics/react-shopping-cart/)

##### В работе использовались: #####
1. Task manager: [Webpack](https://webpack.js.org/)
2. CSS preproccessor: [SASS](http://sass-lang.com/)
3. JS-library: [ReactJS](https://reactjs.org/)

##### Команды проекта: ######
+ yarn run dev - запуск сервера
+ yarn run prod - сборка проекта
